Digital Image Processing course project in SUSTech

This project implements HSV threshold detetcion, Hough circle detection and 3-D reconstruction of ball's trajectory using video captured by two cameras.


package version requirment:
python 3.8
numpy
matplotlib
numba
cuda
opencv

Some python file with word "demo" might be able to run on your computer.

Paths of some addition parameter in the code might need to overwrite according to your computer.
